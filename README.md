# Desarollo Web Integral - API Node

Proyecto backend en Node.js para el curso/ejercicio "Desarollo Web Integral". Contiene una API REST básica para la gestión de recursos y sirve como punto de partida para el desarrollo y despliegue.

## Requisitos

- Node.js (>= 18)
- npm o yarn

## Instalación

1. Clona el repositorio y sitúate en la carpeta del proyecto.
2. Instala dependencias:

```bash
npm install
# o
yarn install
```

## Uso

- Ejecutar en modo producción:

```bash
npm start
```

- Ejecutar en modo desarrollo (con nodemon si está instalado):

```bash
npm run dev
```

## Estructura básica

- `index.js` - punto de entrada (si existe)
- `package.json` - scripts y metadatos
- `README.md` - documentación del proyecto
- `.gitignore` - archivos ignorados por Git

## Notas

- Los commits y mensajes deben estar en español y ser descriptivos.
- Configura variables de entorno en un archivo `.env` no subido al repositorio.
